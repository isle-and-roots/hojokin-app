# 収益モデル・ビジネス戦略

## アプリ概要

**補助金サポート**: AI（Claude）で中小企業の補助金申請書類を自動生成する Web アプリ。
- **技術スタック**: Next.js 16 / React 19 / TypeScript (strict) / Tailwind 4 / Supabase / Claude API
- **補助金データ**: 105件以上（FULL AI対応: JIZOKUKA, IT_DONYU, MONODZUKURI, SHOURYOKUKA, SHINJIGYO_SHINSHUTSU, SEICHOU_KASOKUKA, JIGYOU_SAIKOUCHIKU, SETSUBI_TOUSHI, JINZAI_IKUSEI）

## MK エージェント出力フォーマット

全 MK エージェントは以下の構造で出力:

1. **結論**: 何を作るか（1 文）
2. **ターゲット**: 誰向けか（具体的ペルソナ）
3. **チャネル**: どこで使う / 公開するか
4. **成果物**: 実際のアウトプット
5. **測定方法**: 効果の計測方法
6. **収益インパクト**: 月 4 万円目標への貢献
7. **次のアクション**: 次に動くべき HK / MK エージェント

## ゼロコストチャネル戦略（優先順）

1. **SEO / オーガニック検索**: 補助金検索者は Google で能動検索。既存 26+ ページの SSG 基盤活用
2. **note.com クロスポスト**: 高ドメインオーソリティ、無料、ビジネス層読者
3. **X/Twitter**: 締切リマインダーで自然なエンゲージメント、士業コミュニティとの接点
4. **士業コミュニティ貢献**: Yahoo! 知恵袋、Facebook グループ、業界フォーラムで有益な回答
5. **リファラル / 口コミ**: アプリ内紹介リンク + 紹介特典（双方に Pro 1 ヶ月無料）
6. **ディレクトリ掲載**: BOXIL、ITreview、ProductHunt、Startup DB
