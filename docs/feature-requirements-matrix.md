## 収益構造比較

- **SaaS**: ¥0-9,800/月/ユーザー (セルフサービス)
- **OEM**: ¥50,000-200,000/月/パートナー (カスタマイズ付き)
- **代行**: ¥100,000-300,000/案件 (フルサービス)

## 追加開発工数概算

- OEM対応: 約3-4ヶ月（マルチテナント、ホワイトラベル、管理画面）
- 代行対応: 約1-2ヶ月（クライアント管理、進捗管理、請求）

## 推奨ロードマップ

1. **Phase A** (現在): SaaSとして市場検証・PMF達成
2. **Phase B**: 士業パートナー向けにOEM機能を段階的に追加
3. **Phase C**: 需要があれば代行モデルを検討

## リスク分析

- SaaS: 低リスク、スケーラブル、競合多数
- OEM: 中リスク、高単価、パートナー依存
- 代行: 高リスク、人件費、スケール困難
