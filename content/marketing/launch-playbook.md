## 緊急対応フロー

### AI生成エラーが多発している場合
1. Anthropic API ダッシュボードでエラーログ確認
2. `ANTHROPIC_API_KEY` が本番環境に正しく設定されているか確認
3. API レート制限に達していないか確認
4. 問題が続く場合: X/Twitter でメンテナンス告知

### 決済エラーが発生している場合
1. Polar ダッシュボードでログ確認
2. Webhook の配信ステータス確認
3. Supabase の `user_profiles.plan` を手動で更新（緊急対応）

### 高負荷でサイトが重い場合
1. Vercel Functions のログで bottleneck 確認
2. AI 生成 API の同時実行数を確認
3. Supabase の接続数確認
