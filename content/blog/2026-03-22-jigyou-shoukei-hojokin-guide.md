## 事業承継補助金とは？

**事業承継補助金**は、中小企業・小規模事業者の経営者が、事業承継やM&Aを円滑に進めるために必要な費用を補助する国の支援制度です。親族内承継、従業員承継、第三者承継（M&A）のいずれにも対応しており、後継者不在問題を抱える多くの中小企業にとって重要な制度です。

日本では中小企業経営者の高齢化が進み、後継者不在を理由とした廃業が増加しています。優れた技術や雇用を持つ企業が後継者不足で廃業することを防ぐため、政府は事業承継・M&Aに対する補助を積極的に拡充しています。

## 主な対象者と要件

事業承継補助金は、以下のいずれかのフェーズに対応しています。

| フェーズ | 対象となる事業者 |
|---------|----------------|
| 経営革新型 | 事業承継・M&A後に新しい事業展開に取り組む後継者 |
| 専門家活用型 | M&Aを活用した事業承継に取り組む事業者 |

### 経営革新型の要件

- 中小企業または小規模事業者であること
- 令和元年4月1日以降に事業承継（親族内・従業員・第三者いずれか）を実施または実施予定
- 承継後に新たな取り組みを行うこと（新製品・サービス開発、新規市場開拓、業態転換など）

### 専門家活用型の要件

- 中小企業または小規模事業者であること
- M&Aによる第三者承継を検討・実施している事業者
- M&A仲介業者や専門家（中小企業診断士、M&Aアドバイザーなど）を活用すること

## 補助上限額と補助率

### 経営革新型

| 項目 | 内容 |
|------|------|
| 補助上限額 | 600万円（廃業・再チャレンジの場合は+150万円） |
| 補助率 | 2/3（小規模事業者は3/4） |
| 対象経費 | 設備費、店舗改修費、人件費、広告費、専門家費用など |

### 専門家活用型

| 項目 | 内容 |
|------|------|
| 補助上限額 | 600万円 |
| 補助率 | 2/3 |
| 対象経費 | M&A仲介手数料、デューデリジェンス費用、専門家報酬など |

## 申請の流れ

事業承継補助金の申請は、以下の流れで進めます。

### ステップ1: 事業承継・引継ぎ支援センターへの相談

各都道府県に設置されている**事業承継・引継ぎ支援センター**（無料）に相談することが推奨されています。センターでは、事業承継の計画策定支援や、後継者とのマッチングを行っています。

### ステップ2: 補助金の対象要件の確認

公募要領を確認し、自社が対象要件を満たしているかを確認します。経営革新型か専門家活用型かを判断し、申請する枠を決めます。

### ステップ3: 事業計画の作成

承継後に取り組む事業の計画を作成します。経営革新型では、「承継後にどのような新しい取り組みを行うか」が審査の核心になります。

### ステップ4: 申請書の提出

電子申請システム（GビズID必須）または紙で申請書を提出します。

### ステップ5: 採択後の事業実施・報告

採択通知後に事業を実施し、完了後に実績報告書を提出します。補助金は実績報告が承認されてから振り込まれます。

## 対象となる主な経費

経営革新型では、以下の経費が補助対象になります。

| 経費区分 | 具体例 |
|---------|--------|
| 設備費 | 機械設備、什器備品、PCなど |
| 店舗改修費 | 内装工事、看板設置など |
| 広告費 | 広告掲載料、ウェブサイト制作費 |
| 専門家費用 | 弁護士、税理士、中小企業診断士の報酬 |
| 調査・研究費 | 市場調査費用、試作品製作費 |
| 廃業費（廃業・再チャレンジの場合） | 解体費、残存物撤去費など |

## 採択率を上げるための5つのポイント

### ポイント1: 「承継後の新たな取り組み」を具体的に示す

経営革新型で最も重要なのは、**承継後にどのような革新的取り組みを行うか**です。「前経営者と同じことを続ける」ではなく、新しい販路開拓、新製品・サービスの開発、業務プロセスの改善など、変化のある取り組みを具体的に示しましょう。

### ポイント2: 承継の必然性を明確にする

なぜ事業承継が必要なのか、現経営者の高齢化・健康問題、後継者候補の存在、地域経済への貢献など、承継の背景と必然性を具体的に記載してください。

### ポイント3: 数値目標を設定する

「売上を○%向上させる」「新規顧客を○件獲得する」「従業員を○人維持する」など、具体的な数値目標を設定することで、計画の実現可能性が伝わりやすくなります。

### ポイント4: 専門家・支援機関と連携する

中小企業診断士や税理士、事業承継・引継ぎ支援センターなどの専門機関と連携して計画を策定することで、計画の信頼性が高まります。認定支援機関のサポートは採択に有利に働く場合があります。

### ポイント5: 廃業・再チャレンジ枠も確認する

経営者が自ら廃業した上で、後継者として再び事業を立ち上げる「廃業・再チャレンジ」の場合、廃業費として追加で最大150万円を申請できます。このオプションの活用も検討しましょう。

## M&A補助金（専門家活用型）の具体的な活用場面

専門家活用型は、主に以下のような場面で活用されます。

- 事業を引き継ぎたい経営者が、M&A仲介会社を通じて後継者を探す費用
- 買い手側として、売り手企業の財務・法務デューデリジェンス（DD）を専門家に依頼する費用
- 譲渡交渉や最終契約書の作成を専門家に依頼する費用

M&Aによる第三者承継は仲介手数料が数百万円になることも多く、この補助金で費用の2/3を賄えることは大きなメリットです。

## よくある質問

**Q: 事業承継はいつまでに完了している必要がありますか？**
A: 経営革新型は令和元年4月1日以降に承継した事業者が対象です。承継後に申請することも、承継予定で申請することも可能です。詳細は公募要領で確認してください。

**Q: 親族への承継も対象ですか？**
A: 親族内承継（子どもへの代替わりなど）も対象です。第三者への承継（M&A）に限定されていません。

**Q: 個人事業主も申請できますか？**
A: 申請可能です。小規模事業者の場合、補助率が3/4に引き上げられます（通常は2/3）。

**Q: 承継前の準備費用も補助対象になりますか？**
A: 経営革新型は承継後の取り組みに関する費用が対象です。承継前の費用は原則として対象外です。専門家活用型はM&A実施に関連する専門家費用が対象です。

## まとめ

事業承継補助金は、後継者問題を抱える中小企業が事業の継続と発展を実現するための重要な支援制度です。

- 補助上限は最大**600万円**（廃業・再チャレンジ加算で最大750万円）
- 親族承継・従業員承継・M&Aのいずれにも対応
- 承継後の**新たな取り組み**の具体性が採択のカギ
- **事業承継・引継ぎ支援センター**への相談から始めるのがおすすめ

事業承継は早めに動くほど選択肢が広がります。承継前の段階から情報収集と計画策定を始めましょう。
