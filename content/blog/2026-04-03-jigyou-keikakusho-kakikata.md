## 補助金の採択率は「事業計画書の質」で決まる

補助金の審査では、書類に不備がなく、申請要件を満たしていることが前提です。その上で審査員が評価するのは**事業計画書の内容と説得力**です。

多くの補助金では採択率が30〜50%程度であり、「要件を満たした申請書の中から採択先を絞り込む」という競争になっています。要件を満たしているだけでは不十分で、他の申請者より「説得力のある事業計画書」を書く必要があります。

この記事では、中小企業診断士の観点から、採択率を高める事業計画書の書き方を解説します。

## 審査員が事業計画書で見ているポイント

補助金の審査基準は公募要領に記載されていますが、審査員が評価する内容は共通しています。

### 1. 課題の明確さ

「なぜこの補助金が必要なのか」が明確か。曖昧な課題設定では、「本当にこの事業に補助金が必要なのか」と判断されます。

### 2. 取り組みと課題の整合性

課題に対して、提案する取り組みが論理的に対応しているか。「課題Aに対して取り組みBを行う」という因果関係が明確かどうかが問われます。

### 3. 数値目標の具体性と根拠

「売上が向上する」ではなく「売上が○%向上する（根拠: ○○）」という形で、数値目標とその根拠がセットになっているか。

### 4. 実現可能性

人材・設備・資金・技術などの観点から、計画が現実的に実行できるか。

### 5. 補助事業終了後の継続性

補助期間が終わった後も、事業として自立できる計画になっているか。

## 採択される事業計画書の構成

### 基本構成: 「現状→課題→取り組み→効果」

採択される事業計画書は、以下の流れで書かれています。

```
1. 現状（自社の状況を客観的に説明）
    ↓
2. 課題（現状のどこに問題があるか）
    ↓
3. 取り組み（補助事業の内容・方法）
    ↓
4. 効果（取り組みによってどのような変化が生まれるか）
    ↓
5. 継続性（補助終了後の事業計画）
```

この流れが一貫して保たれている申請書は、審査員にとって「読みやすく、論理的」に映ります。

### セクション別の書き方ポイント

#### 「企業概要」セクション

**書くべき内容:**
- 業種・事業内容（主力商品・サービス、顧客層、営業エリア）
- 創業年・従業員数・年商（概算でも可）
- 強み・差別化要因

**NG表現の例と改善:**

NG: 「地域に根ざした企業として、長年にわたりお客様に信頼されています。」

OK: 「創業23年の〇〇製造業（従業員12名、年商1.5億円）。地域の飲食店・小売店向けに食材加工サービスを提供。リピート率85%で、地域内シェア約30%を占める。」

数値と具体性が「企業の実態」を伝えます。抽象的な表現は審査員に響きません。

#### 「現状・課題」セクション

これが事業計画書の核心です。「課題が明確で深刻か」「その課題がこの補助金の目的と合っているか」を見られています。

**書くべき内容:**
- 現在の業務の実態（何をどのように行っているか）
- そこにどのような問題・課題があるか
- 課題によって生じている損失（時間・コスト・機会損失）

**NG表現の例と改善:**

NG: 「業務が非効率で、改善の余地があります。」

OK: 「受注・出荷・在庫管理を手書き伝票で行っており、月末の集計・照合作業に経営者自身が月30時間を費やしている。人手不足により欠品が月平均3件発生し、月間15万円程度の機会損失が生じている。また、Excelでの在庫管理はリアルタイム性が低く、適正在庫量の把握が困難な状況にある。」

#### 「取り組み・実施計画」セクション

**書くべき内容:**
- 何を（What）: 具体的な設備・ツール・施策の内容
- なぜ（Why）: この取り組みを選んだ理由
- どのように（How）: 実施方法・スケジュール
- 誰が（Who）: 担当者・体制

**ガントチャートの活用:**

実施計画はガントチャート（スケジュール表）形式で示すと視覚的に伝わりやすく、審査員にも評価されます。

| 月 | 1ヶ月目 | 2ヶ月目 | 3ヶ月目 | 4〜6ヶ月目 |
|----|--------|--------|--------|----------|
| システム選定・契約 | ■ | | | |
| データ移行・設定 | | ■ | | |
| 社内トレーニング | | | ■ | |
| 本格運用開始 | | | | ■■■ |

#### 「期待される効果」セクション

**書くべき内容:**
- 定量的な効果（数値で示す）
- 定性的な効果（数値化しにくいが重要な変化）
- 効果の発現時期

**NG表現の例と改善:**

NG: 「業務効率が大幅に向上し、売上増加が期待されます。」

OK: 「在庫管理システムの導入により、月30時間の集計・照合業務を月8時間（73%削減）に圧縮する。捻出した22時間/月を営業活動に充てることで、新規顧客を月1〜2件獲得し、6ヶ月以内に月間売上を現在の600万円から660万円（10%増）に引き上げることを目標とする。また、欠品発生件数をゼロに改善し、月15万円程度の機会損失を解消する。」

## 数値の使い方：「根拠のある数値」が採択を決める

### 数値は「現状」と「目標」の両方を示す

| 項目 | 現状 | 目標 | 削減率/増加率 |
|-----|-----|-----|------------|
| 月間経理業務時間 | 25時間 | 7時間 | 72%削減 |
| 月間受注ミス件数 | 4件 | 0件 | 100%解消 |
| 月間売上 | 1,200万円 | 1,440万円 | 20%増加 |

### 数値の根拠を示す方法

「根拠なき数値」は審査員に信頼されません。数値の根拠を示す方法として、以下のアプローチが有効です。

**実績データ:**
「現在、請求書発行に月25時間を費やしている（6ヶ月分の業務記録より）」

**業界平均・先行事例:**
「同規模の小売業でのシステム導入事例では、在庫管理工数を平均68%削減している（中小企業庁 IT導入補助金 活用事例集より）」

**計算根拠:**
「受注処理の自動化により、1件あたりの処理時間が15分から5分に短縮（10分削減）。月間受注件数150件で換算すると、月25時間の削減となる」

## 補助金別・事業計画書作成のポイント

補助金の種類によって、審査で特に重視されるポイントが異なります。

### 小規模事業者持続化補助金

**重視ポイント:** 「販路開拓・売上拡大」の具体性

販路開拓のターゲット（誰に）、チャネル（どこで・どうやって）、見込み効果（どれだけ）を具体的に記載することが重要です。

「Webサイトを作って新規顧客を獲得する」ではなく、「どのターゲット層に対して、どのようなWebコンテンツを発信し、月○件の問い合わせを獲得するか」まで具体化しましょう。

### ものづくり補助金

**重視ポイント:** 「革新性・技術的新規性」の明確な説明

既存の製品・サービスからどの点が革新されているか、技術的な新規性がどこにあるかを、専門用語を交えながらも分かりやすく説明することが必要です。審査員には技術に詳しくない方も含まれるため、「この技術が革新的である理由」を非専門家にも伝わる言葉で説明しましょう。

### 事業再構築補助金

**重視ポイント:** 「新規性・転換の明確さ」と「市場の成長性」

既存事業から新規事業への転換が「単なる新商品追加ではなく、事業の本質的な転換である」ことを示す必要があります。また、参入する市場の成長性についても言及することで、「この事業が将来的にも継続・成長できる」という説得力が増します。

## 事業計画書作成でよくある質問

### Q. 文字数はどのくらい書けばいいか？

補助金によって指定文字数は異なりますが、一般的には「指定文字数の80〜100%を埋める」ことが目標です。文字数が極端に少ない申請書は「内容が薄い」と見なされます。

### Q. 図表は使ってよいか？

ほとんどの補助金で図表の活用は可能です（電子申請の場合、PDF添付のケースが多い）。ガントチャート・フロー図・比較表などを活用することで、複雑な情報が視覚的に伝わりやすくなります。

### Q. 素人が書いても採択されるか？

採択されます。ただし、「素人だから粗くてよい」ではなく、「論理の一貫性と数値の具体性」に集中することが重要です。文章の上手さより、「課題→取り組み→効果」の論理が一貫しているかどうかが評価されます。

## まとめ: 採択率を上げる事業計画書の5つの鉄則

1. **「現状→課題→取り組み→効果」の流れを一貫させる**
2. **すべての主張を数値で裏付ける**（数値がない文章は避ける）
3. **課題はできるだけ具体的・深刻に書く**（「改善の余地がある」ではなく「月○時間・○万円の損失が生じている」）
4. **取り組みの選定理由を明確にする**（なぜこの設備・ツール・施策を選んだか）
5. **補助終了後の継続性・発展性を示す**（補助金なしでも事業が続くことを証明する）
